How it works

OpenClaw runs silently in the background on your laptop/PC as a daemon. You interact with it through messaging apps (Telegram, WhatsApp, Discord) on your phone, or through the browser dashboard. It uses an AI model via API key to understand your requests and execute tasks.

📱 Your phone (Telegram / WhatsApp)
↕
⚙️ OpenClaw daemon — running on laptop
↕
🤖 AI model API (Groq / Gemini / OpenAI)
Where things live

Installed on: Your laptop/PC (runs as a background process)
Used from: Phone via Telegram/WhatsApp, or browser at 127.0.0.1:18789
Config stored at: ~/.openclaw/ (Linux/Mac)  ·  C:\Users\YourName\.openclaw\ (Windows)

For daily use, OpenClaw works best on a machine that stays on 24/7 — a desktop, Mac Mini, or a cheap VPS. On a laptop it pauses when you close the lid.

Install & setup

One-time setup — takes about 5 minutes

1
Install Node.js (if not already)

Download from nodejs.org — version 22 or newer required.

2
Install OpenClaw via npm
npm install -g openclaw@latest

Or with pnpm: pnpm add -g openclaw@latest

3
Run the onboarding wizard
openclaw onboard --install-daemon

This walks you through connecting your AI provider (API key) and messaging apps. The --install-daemon flag makes OpenClaw auto-start on every reboot.

4
Open the dashboard

Go to 127.0.0.1:18789 in your browser. Bookmark it for quick access!

5
Add a channel

Dashboard → Channels → Add Channel → follow setup for Telegram or WhatsApp.

Telegram bot setup

Easiest way to use OpenClaw from your phone

1
Create a bot via BotFather

Open Telegram → search @BotFather (must have ✅ blue verified tick) → type /newbot → give it a name (e.g. "My Assistant") → give it a username ending in "bot" (e.g. myassistant_bot) → copy the token it gives you.

Token looks like: 7483920156:AAFkdjsf83jdKSlfj39dLSKfjd83jdKS
2
Add the bot to OpenClaw

Dashboard → Channels → Add Channel → Telegram → paste the bot token → Save.

3
Approve yourself (security step)

Message your bot /start — it will show your Telegram user ID. Run this in terminal (replace XXXXXXXX with your actual ID):

openclaw pairing approve telegram XXXXXXXX

Or go to Dashboard → Instances → approve from there.

4
Start chatting!

Type /start again in Telegram — it should reply. Then just talk to it naturally in plain language.

If BotFather blocks new bot creation: switch between WiFi and mobile data, wait 10–15 minutes and retry, or use a VPN temporarily just to create the bot.
Regenerate token if needed

BotFather → /mybots → select your bot → API Token → Regenerate. Paste the new token in OpenClaw Channels settings.

WhatsApp setup

Links your real WhatsApp number via QR code scan

WhatsApp has no official free bot API. OpenClaw links your actual WhatsApp number using an unofficial method. Small risk of account restriction. You can run both Telegram and WhatsApp at the same time.
1
Open Channels in the dashboard

Go to 127.0.0.1:18789 → Channels → Add Channel → WhatsApp

2
Scan the QR code

A QR code appears on screen. On your phone: WhatsApp → Settings → Linked Devices → Link a Device → scan the QR code.

3
Test it

Send any message — OpenClaw should respond via WhatsApp.

Telegram is easier and safer. Use WhatsApp only if you specifically need it. Always keep Telegram running as a backup channel.

How to replace an expired/broken API key
1
Get a new key

Go to your provider's website → create a new API key → copy it.

2
Update in dashboard

127.0.0.1:18789 → Agents → your agent → find the API key field → paste new key → Save.

3
Restart
openclaw restart
If you see "auth_permanent issue" in logs — the key is permanently rejected. Get a brand new key from the provider's website, don't just regenerate the same one.

Errors & fixes

Every error we hit — with exact solutions. Click to expand.

401Telegram Unauthorized — webhook/setMyCommands fail
+
telegram deleteWebhook failed: Call to 'deleteWebhook' failed! (401: Unauthorized)
telegram setMyCommands failed: 401 Unauthorized

Cause: Bot token is wrong, invalid, expired, or has extra spaces.
Fix: BotFather → /mybots → your bot → Regenerate Token → paste it fresh in OpenClaw Channels settings. Make sure no spaces before/after token.

LIMITGemini API rate limit reached
+
API rate limit reached. Please try again later.
auto-restart attempt 4/10 in 42s

Cause: Free Gemini tier allows ~15 requests/minute. You exceeded it.
Fix: Wait 1–2 minutes (OpenClaw auto-retries up to 10 times). Or switch to Groq API for much better free limits.

AUTHGoogle auth_permanent failure — skipping all models
+
Provider google has auth_permanent issue (skipping all models) (auth_permanent)

Cause: API key is deleted, completely invalid, or expired. OpenClaw permanently stops trying Google.
Fix: Get a brand new key from aistudio.google.com → update in Agents settings → run openclaw restart.

CONFIGConfig validation failed — Unrecognized key
+
Config validation failed: <root>: Unrecognized key: "GOOGLE_API_KEY"

Cause: Wrong config key name used in terminal command.
Fix: Don't use terminal for API key updates. Use the dashboard: 127.0.0.1:18789 → Agents → your agent → update the key field directly. Or run openclaw config list to see all valid key names first.

ACCESS"Access not configured" when messaging bot
+
Access not configured. Your Telegram user ID: XXXXXXXX
Ask bot owner: openclaw pairing approve telegram XXXXXXXX

Cause: This is a security feature — OpenClaw requires you to explicitly approve each user.
Fix: Run openclaw pairing approve telegram XXXXXXXX in terminal with your actual ID. Then type /start again in Telegram.

BLOCKBotFather blocks new bot creation
+

Cause: New Telegram account age restriction, too many attempts, or regional IP block.
Fix options:
· Wait 10–15 minutes and try again
· Switch between WiFi and mobile data (different IP)
· Use a VPN temporarily just to create the bot
· Use Discord or the web dashboard at 127.0.0.1:18789 instead

Pro tips

Things that make OpenClaw much easier to use daily

📌 Bookmark the dashboard

Add 127.0.0.1:18789 to your browser bookmarks. You'll use it constantly for settings, logs, and debugging.

⚡ Use Groq instead of Gemini

Groq's free tier is faster and has much better rate limits than Gemini. Get your key at console.groq.com — takes 2 minutes and no credit card needed.

🔄 Keep Telegram as backup

Even after setting up WhatsApp, keep the Telegram bot running. If WhatsApp has issues, Telegram is always there as a fallback.

🔑 API key = kill switch

If OpenClaw ever behaves unexpectedly, delete your API key from the provider's website. Without a valid key, it can't do anything — even if the process is still running.

🎯 Be specific with requests

Instead of "check email", say "check Gmail and tell me if there are any urgent unread emails from the last 24 hours". More specific = much better results.

🔁 Auto-start is on by default

If you installed with --install-daemon, OpenClaw starts automatically on every reboot. Just open the browser dashboard — no need to manually start it.

🔍 Watch live logs when debugging
openclaw logs --follow

Shows exactly what OpenClaw is doing in real time. Very helpful when something isn't working.

💡 Phone vs browser

Use Telegram on your phone for daily chatting on the go. Use the browser dashboard when you want to change settings, check usage stats, or debug issues.
